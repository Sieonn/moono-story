## MoonoStory
캐릭터 무너를 간단한 미션 참여를 통해서 알아볼 수 있는 인터렉티브 사이트 입니다.

### 🧶 Skill

- frontend
  - react
  - Typescript
---

> 프로젝트 실행방법
> 1. 터미널을 키고 npm install을 입력한다.
> 2. 2. npm start 로 실행시킨다.
